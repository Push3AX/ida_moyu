# ida_moyu

摸鱼🐟用的IDA插件，在Pseudocaode风格的窗口中浏览文本网页或者文件，看起来就像在用IDA工作。

### 安装

将 `ida_moyu.py` 复制到 IDA 的插件目录：

- macOS 下：`~/.idapro/plugins/`
- Windows / Linux 下：`<IDA install dir>/plugins/`

重启 IDA ，通过`Edit -> Plugins -> ida_moyu`打开插件：

![ida_moyu](https://raw.githubusercontent.com/Push3AX/ida_moyu/refs/heads/main/ida_moyu.png)

---

A simple IDA plugin for browsing text web pages, or files in a Pseudocode-style window, so it looks like you are working in IDA.

### Installation

Copy `ida_moyu.py` into IDA's plugin directory:

- macOS: `~/.idapro/plugins/`
- Windows / Linux: `<IDA install dir>/plugins/`

Restart IDA, then open the plugin from `Edit -> Plugins -> ida_moyu`:

![ida_moyu](https://raw.githubusercontent.com/Push3AX/ida_moyu/refs/heads/main/ida_moyu.png)
